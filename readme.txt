Super offline browser for wordpress.



== Description ==

This is one of the best plugin which creates offline cache  and increases loading time and speed of your wordpress website.



== Installation ==


1.Upload WP Offline Browser Plugin folder to your's wordpress plugin directory.

2.Activate the  WP Offline Browser Plugin.

4.Go to WP Offline Browser setting page and click generate cache button.




== Frequently Asked Questions == 


= The plugin is not working ? =

You may need to add a custom file type to your web server or .htaccess configuration.
add this line to your config file:

AddType text/cache-manifest .manifest 


= The site updates are not showing =

If you’re adding or removing resources , you’ll have to regenerate cache by clicking generate cache button on plugin setting page.
